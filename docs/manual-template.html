세계로 무역 게임 — 시작 가이드

교사가 방을 열면 학생들이 국가의 대표가 되어 협상하고, 함께 의사결정하며, 자원을 사고팔고 가공해 이익을 남기는 교실용 무역 게임입니다. 이 문서는 화면을 그대로 따라가며 방을 만들고 게임을 진행하는 법을 보여줍니다.

시작하기 전에 👩‍🏫 교사 가이드 🧑‍🎓 학생 가이드 진행 팁

시작하기 전에

첫 화면을 열면 게임 방법을 6장으로 설명하는 인트로가 자동으로 뜹니다. 그중 하나는 이 게임의 핵심인 모둠의 의사결정을 보여줍니다 — 국가 안에서 거래를 보내거나 받아들일 때 모둠원 과반수 찬성이 필요합니다.

첫 화면 교사와 학생, 두 가지 입장 방법
인트로 3/6 "모둠이 함께 결정해요" 슬라이드

교사 가이드

6단계

방을 만들고, 학생을 초대하고, 턴을 진행하는 전체 흐름입니다.

01

무대와 난이도 정하기

세계 국가(14개국)와 한국 지역 중 무대를 고릅니다. 한국을 고르면 시/군/구, 시/도, 또는 충북 11개 시/군 중 단위를 한 번 더 선택합니다. 난이도는 다루는 자원의 범위를 정합니다 — 쉬움은 원자재만, 어려움은 완제품까지 갑니다.

1단계 무대 선택
한국 선택 시 시·군·구 / 시·도
난이도 쉬움 / 보통 / 어려움
02

참가 국가 수 정하기

학생 모둠 수에 맞춰 참가 국가 수를 −/+로 조절하세요. 대표 국가가 실제 특산품과 함께 자동으로 채워지고, 필요하면 카드 안에서 이름·특산품·생산량을 직접 바꿀 수 있습니다.

⚖️ 균형 적절 / ⚠️ 평균보다 높음·낮음 배지가 국가마다 떠요. 한 국가의 턴당 생산 가치가 다른 국가들 평균보다 많이 차이 나면 경고가 뜨니, 생산량을 조금 늘리거나 줄여서 맞춰 주세요. 모든 국가는 시작 자금도 똑같이 받고, 턴당 생산량만큼 자원을 미리 가지고 시작하므로 이 배지만 잘 맞추면 출발선이 공평해집니다.
2단계 국가 수 조절 + 특산품 자동 채움
03

게임 규칙 정하기

시작 자금, 목표 턴 수, 그리고 모둠 승인 방식을 정합니다.

팀 투표(추천)를 켜면 거래를 보내거나 받아들일 때 모둠원 과반수 찬성이 필요해요. 모두가 참여하는 의사결정을 경험하게 하려면 이 옵션을 켜 두세요.
💰 시작 자금은 1원 · 십원 · 백원 · 천원 · 만원 단위로 조절할 수 있어요. 1학년처럼 아주 어린 학급이라면 1원이나 십원 단위까지 내려서 10원, 50원처럼 계산하기 쉬운 숫자로 맞춰 주세요. 모든 국가가 정확히 같은 금액을 받습니다. 단위 버튼을 누르든 숫자를 직접 입력하든, 시작 자금에 맞춰 모든 자원의 시세도 자동으로 같은 비율로 바뀝니다 — 화면에 바로 미리보기가 떠요.
3단계 시작 자금 · 목표 턴 · 승인 방식
04

학생 초대하기

방이 만들어지면 큰 화면에 QR 코드와 6자리 코드가 함께 뜹니다. 학생이 QR을 스캔하면 코드 입력 없이 이름만 넣고 바로 국가를 고를 수 있습니다.

로비 QR · 코드 · 참가 현황
💡 프로젝터·TV 화면에 이 화면을 띄우고, 학생들이 각자 폰으로 QR을 스캔하게 하면 가장 빠릅니다.
05

게임 진행하기

2개 이상의 국가에 학생이 들어오면 게임 시작이 활성화됩니다. 학생들이 협상하는 동안 교사는 거래 현황을 지켜보고, 이벤트 카드로 시세를 흔들다가, 협상이 충분히 끝났을 때 턴 진행하기를 누릅니다.

대시보드 국가 현황 + 턴 진행 버튼
거래 현황 합의된 거래 목록
이벤트 카드 랜덤 뽑기 또는 직접 선택 · 128종
턴 진행을 누르면 ① 대출·상환 → ② 거래 체결 → ③ 특산품 생산 → ④ 시장 구매 → ⑤ 제작 → ⑥ 시장 판매 순서로 한꺼번에 처리됩니다. 되돌릴 수 없으니 협상이 충분히 끝난 뒤 눌러 주세요.
🌍 세계 무역시장 · 🏦 세계 은행은 학생이 언제든 즉시 신청할 수 있어요 (다른 국가와 협상할 필요 없음). 다만 실제로 돈·자원이 오가는 건 교사가 턴 진행을 눌렀을 때예요 — 신청만 미리 쌓이는 구조라 변조 감지 기능은 그대로 안전하게 작동합니다. 세계 국가 모드가 아니어도 똑같이 쓸 수 있어요 — 한국 지역 모드(시/군/구·시/도·충북 시/군 모두)에서는 같은 기능이 🏬 거래소 · 🏦 한국은행이라는 이름으로 나타날 뿐, 즉시 매매·대출이 되는 원리는 똑같습니다.
06

결과 확인하기

목표 턴에 도달하거나 게임 종료를 누르면 최종 순위와 전체 기록을 볼 수 있습니다.

결과 최종 순위표 + 전체 진행 기록
📥 기록 다운로드 버튼을 누르면 턴 진행 기록과 모둠별 회의 채팅까지 시간 순서대로 합쳐서 텍스트 파일로 받을 수 있어요. "기록" 탭에서는 게임 중에도 언제든 같은 내용을 확인할 수 있고, 게임이 끝난 뒤 결과 화면에서도 그대로 남아 있어 평가 자료로 쓰기 좋습니다.

학생 가이드

7단계

입장부터 협상, 가공과 수출, 결과 확인까지의 흐름입니다.

01

입장하기

선생님이 보여준 QR을 스캔하면 코드가 자동으로 채워집니다. QR이 없다면 6자리 코드를 직접 입력하세요. 이름은 모둠원이 알아볼 수 있게 적습니다.

입장 코드와 이름 입력 (QR 스캔 시 코드는 자동 입력)
02

국가와 역할 고르기

비어 있는 국가를 고르거나, 친구가 이미 들어간 국가를 골라 같은 모둠이 되세요. 인원 제한은 없습니다. 역할은 참고용이라 모든 활동은 누구나 할 수 있어요.

국가 선택 특산품이 표시된 국가 카드
역할 선택 무역대표 · 생산부장 · 재무관 · 정보분석가
⚠️ 역할은 실제 게임에 아무 영향을 주지 않습니다. 무역대표든 재무관이든 관계없이 모둠원 누구나 거래를 만들고, 제작을 예약하고, 투표할 수 있어요. 역할은 순전히 모둠 안에서 할 일을 나눠 맡기 위한 참고용 이름표일 뿐이니, 마음에 드는 걸로 자유롭게 고르면 됩니다.
03

우리 국가 화면 살펴보기

매 턴 자동으로 생산되는 특산품, 창고, 모둠원 목록을 한눈에 봅니다. 게임을 시작하자마자 첫 턴 생산량만큼 자원을 이미 가지고 있어요 — 다른 국가가 거래를 받아 줄 때까지 기다리지 않고 바로 협상을 시작할 수 있습니다.

🎯 우리가 원하는 물자 — 우리 국가가 특히 필요로 하는 자원이에요. 이 물자를 사서 모으면 총자산이 30% 더 오릅니다. 국가마다 원하는 게 다르니 서로 무역할 이유가 생깁니다.
🏦 세계 은행도 이 화면에 있어요. 현금이 부족하면 언제든 대출을 신청할 수 있고 (이자 10%), 여유가 생기면 갚을 수 있어요. 대출·상환도 턴이 진행될 때 실제로 반영됩니다.
우리 국가 생산 · 원하는 물자 · 창고 · 모둠
04

무역하기 — 협상하고 함께 결정하기

"다른 국가" 탭에서 누가 무엇을 가졌고 무엇을 원하는지 확인한 뒤, "무역" 탭에서 팔 자원과 수량, 금액을 정해 제안을 보냅니다.

다른 국가 보유 자원과 원하는 물자 확인
제안 만들기 시세 대비 가격 안내까지 표시
🗳️ 의사결정이 여기서 일어납니다. 팀 투표가 켜져 있으면 제안을 보내기 전에 우리 모둠원 과반수가 찬성해야 하고, 받은 제안을 수락할 때도 상대 모둠의 과반수가 찬성해야 해요.
🔒 같은 자원을 두 번 파는 일은 없어요. "무엇을" 목록에는 실제로 지금 팔 수 있는 수량만 나와요 — 모둠원이 이미 다른 나라에 제안했거나 제작·세계시장 판매에 쓰기로 예약한 만큼은 미리 빠져 있어서, 여러 명이 동시에 같은 자원을 이중으로 약속할 걱정이 없습니다.
보낸 제안 모둠 찬성을 기다리는 중
받은 제안 찬성 · 반대 투표
05

이벤트 카드 확인하기

선생님이 이벤트 카드를 발동하면 시세가 바뀌고, 화면 위에 속보 배너로 뜹니다. 어떤 자원이 오르고 내렸는지 확인하고 전략을 바꿔보세요.

속보 시세 변동 알림
06

가공하고 세계시장에서 사고팔기 — 진짜 이익은 여기서

"제작" 탭에서 원자재를 조합해 더 비싼 물건을 만드세요. 재료가 모자라면 그 자리에 뜨는 🛒 사기 버튼으로 바로 세계시장에서 살 수 있어요. 다 만든 가공품·완제품은 📈 시세 탭의 🌍 세계 무역시장에서 시세보다 웃돈(+15~30%)을 받고 팔 수 있습니다.

제작소 재료 → 완성품, 예상 이익까지 표시
🌍 세계 무역시장은 다른 국가와 협상 없이 즉시 사고팔 수 있어요. 다만 한 자원을 계속 사면 값이 오르고 계속 팔면 값이 내려가서, 세계시장만 쓰기보다 다른 국가와 직접 협상하는 편이 더 유리할 때가 많아요. 신청도 턴 진행 때 한꺼번에 처리됩니다.
07

결과 확인하기

게임이 끝나면 순위와 트로피, 우리 모둠의 활동 내역을 볼 수 있습니다.

결과 최종 순위 + 트로피 + 모둠 활동

진행 팁

🌱

처음이라면 쉬움부터

제작 없이 사고팔기에만 집중하는 난이도예요. 무역 자체에 먼저 익숙해지게 하세요.

🗳️

팀 투표는 켜 두세요

혼자 결정하게 하면 조용한 학생은 참여하지 않습니다. 과반수 찬성 규칙이 모두를 끌어들여요.

⚠️

값이 이상하면 경고가 떠요

학생이 개발자도구로 값을 바꾸면 교사 화면에 경고와 "되돌리기" 버튼이 나타납니다.

🏭

가공은 어려움에서만

보통 난이도는 1차 가공품까지, 어려움은 완제품과 세계시장 판매까지 열립니다. 학년에 맞춰 고르세요.

📱

QR은 화면에 크게

교실 TV·프로젝터에 로비 화면을 띄우면 학생들이 코드를 손으로 입력할 필요가 없습니다.

⏱️

턴 진행은 신중하게

한 번 누르면 되돌릴 수 없어요. 협상이 충분히 끝났는지 거래 현황 탭에서 먼저 확인하세요.

🌍

세계시장은 보험, 협상은 본게임

세계시장은 언제든 즉시 사고팔 수 있지만 많이 살수록 비싸지고 많이 팔수록 싸져요. 진짜 이득은 다른 국가와의 직접 협상에서 나온다는 걸 알려 주세요.

🏦

대출은 이자 10%

세계 은행 대출은 급할 때 유용하지만 갚을 때 10% 이자가 붙어요. 빚을 계획 없이 늘리면 총자산이 오히려 줄어든다는 걸 경험하게 해 주세요.
